Optimisation

To decrease image loading times, all image assets were converted to .webp format from .jpg

This reduced image sizes by roughly 50% and produced a marked improvement in site loading times on first visit.

Debugging

Debugging was done with the W3 HTML and CSS validators.

According to the W3 HTML Validator, no errors were found in my html files.

According to the W3 CSS Validator, 1 warning was given:

This refers to the blacked out solutions to the quiz in the quiz page. The colors being the same is intentional as the answer should not be visible until hovered over.

Accessibility Testing

Accessibility testing was done with the Web Accessibility Evaluation Tool

Contrast on the site passed WCAG requirements with a contrast ratio of 8.59:1. Falsely flagged contrast issues were found once again in the quiz section, which can be safely ignored as it is not a bug but a feature.

In every other case, the outputs were positive and no errors were found.
