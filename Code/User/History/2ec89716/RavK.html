Accessibility

Efforts have been made to make this website as accessible as possible. If you think that your accessibility needs are not being met, please contact me.

Font Size

Regarding font size, the site has been tested up to 300% zoom scale and retained responsive functionality. On PC or laptop, use your browser's zoom function with CTRL + (Windows) and CMD + (Mac)

Readability

High contrast colors have been used in purple and yellow for hover effects. This is an attempt to mitigate color blindness causing issues navigating the site. The website is completely WCAG compliant with a contrast of 9.6:1 at its worst.

Navigation

On mobile devices, large blocks have been used for the navbar to reduce risk of fat-fingering the wrong page. The navbar is at the top of the page and can be easily found.
