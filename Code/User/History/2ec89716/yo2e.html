Accessibility Efforts

This website has been made accessible to the best of my ability. If you think that your accessibility needs are not being met, please contact me and give feedback.

Font Size

A comfortable reading size has been selected, but if insufficient, the site has been tested up to 300% zoom scale and retained responsive functionality. On desktops or laptops, use your browser's zoom function with CTRL + (Windows) or CMD + (Mac). On mobile, use your browser's accessibility features or enter reader mode (for apple devices).

Readability

A high contrast color scheme has been used for the navbar and website elements in general. Tools in the form of extensions can be used to increase contrast even further, like the high contrast extension.

Navigation

The navbar can always be found by scrolling to the top of the page. It will not be hidden in any case. On mobile devices, a more user friendly navbar is used instead of an inline one.

Text to Speech

Orca and the Pericles Browser Extension work with most of the content on the page. Any bugs or content skipped over can be reported via contact.
