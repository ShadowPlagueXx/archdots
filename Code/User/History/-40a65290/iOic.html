Introduction

My approach to the design of this website was to create a lightweight website with minimal clutter. It must be easy to navigate for users, and the information conveyed must be clear.

~Colours~

I really wanted to make the website with a little 'retro' feel, So I went for a beige coloured background and having thick borders around all of my images. The colour also acts as a blue light filter, making it easier on the reader's eyes. For UI elements, purple felt appropriate as it is the primary color of the university.
